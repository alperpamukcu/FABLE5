## 6. VIP / CRITIC CUSTOMERS (BOSSES)

One VIP per Night, drawn from a pool; each imposes a rule for that order. 20 at launch, examples:

| VIP | Rule |
|---|---|
| **The Teetotaler** | Spirits score 0 Flavor this order. |
| **The Purist** | Only your first Mix counts; others score 0. |
| **The Allergic** | All Garnish cards are debuffed. |
| **The Snob** | Only Recipes level 2+ score. |
| **The Regular's Ghost** | Every Mix must be a different Recipe. |
| **The Health Inspector** | Rail size −3 this order. |
| **The Rush Hour** | 30-second real-time limit per Mix (only VIP with a timer). |
| **The Critic** (Night 8 finisher, always) | Target ×1.5 and one random Ingredient Type is debuffed, revealed at start. |

VIP rules are always displayed on the customer's speech bubble and order ticket. Defeating a VIP grants a bonus tip ($5 base).
